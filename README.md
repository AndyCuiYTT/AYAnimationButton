# AYAnimationButton
自定义动画 Button

> ## AYAnimationButton 点击水波扩散效果。
> 该控件继承于 UIView ，可添加子控件完成自定义。
> 通过实现 ay_setBtnClickBlock 方法完成点击事件设置。

> ## AYImagesAnimationButton 自定义动画按钮
> 继承于 UIImageView 有 UIImageView 的全部属性
